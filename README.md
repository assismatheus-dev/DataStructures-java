### ðŸ”¹ AVL Tree

A self-balancing Binary Search Tree where:
- The height difference between left and right subtrees
  is at most 1
- Rotations are performed automatically after insertions
  and removals to maintain balance

Implemented features:
- Insertion with rebalancing
- Removal with rebalancing
- Rotations (Left, Right, Left-Right, Right-Left)
- Tree traversals
